LOGIN
[L｜LOGIN] ** ** **
描述：
	L 或 LOGIN 命令用于登录，需要三个参数，分别是用户名、密码、一个整数
示例：
	L test test 5
	LOGIN test test 5
	
LOGINOUT
[LO｜LOGINOUT]
描述：
	LO 或 LOGINOUT 命令用于退出登录
示例：
	LO
	LOGINOUT
	
OUT
[O｜OUT]
描述：
	O 或 OUT 命令用于退出系统
示例：
	O
	OUT
	
PUT
[P｜PUT]
描述：
	P 或 PUT 命令用于新增或更新存储信息
示例：
	P
	PUT
	
REMOVE
[R|REMOVE] [**]
描述：
	R 或 REMOVE 命令用于删除存储的信息，有参数删除指定数据，否则删除所有数据
示例：
	R
	REMOVE
	R CODE
	REMOVE CODE
	
FIND
[F|FIND] [**]
描述：
	F 或 FIND 命令用于查看存储的信息，有参数查看指定数据，否则查看所有数据
示例：
	F
	FIND
	F CODE
	FIND CODE
